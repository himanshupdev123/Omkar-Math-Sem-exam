Question 5 – Partial Derivatives Proof (sin⁻¹)

Time: 00:00

If u = sin⁻¹ \left( \frac{x + 2y + 3z}{x^8 + y^8 + z^8} \right), show that

x u_x + y u_y + z u_z = -7 \tan u

(Most repeated proof in mid-sem papers, CO2)

Question 6 – Partial Derivatives Proof (tan⁻¹)

Time: 00:00

If u = \tan^{-1} \left( \frac{x^3 + y^3}{x - y} \right), show that:

i) x u_x + y u_y = \sin(2u)

ii) x^2 u_{xx} + 2xy u_{xy} + y^2 u_{yy} = 2 \cos(3u) \sin(u)

(Appears in almost every mid-sem QB, CO2)
